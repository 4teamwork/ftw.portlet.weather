Overview
========

``ftw.portlet.weather`` provides the sf meteo flash forcast from sf.meteo.ch


Install
======

- Add ``ftw.portlet.weather`` to your buildout configuration

::

  [instance]
  eggs =
    ftw.portlet.weather

- Run buildout

- Install ``ftw.portlet.weather`` in portal_setup
